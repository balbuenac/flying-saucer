# Introduction #

On February 16, 2011, all open issues on our original java.net hosting site for the project (http://xhtmlrenderer.dev.java.net) were migrated via script to the Google Code issue tracker. The table below summarizes the relationship between old and new issue IDs. Using the Google Code client APIs, I was able to add new issues in GC programmatically, but could not update the old issue tracker programmatically. Hence the new issues include a hyperlink back to the original one, but not vice-versa.
